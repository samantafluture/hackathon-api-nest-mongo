# Hackathon API

Simple Nest.js API for create, update, delete and read infos about Hackathons around the globe.

Built with:
- Nest.js
- MongoDB/Mongoose

Deploy in Heroku.

## Features

- Basic CRUD operations
- No auth routes
- Model with: name, description, location, price, url and date

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
